# Portfolio — Dharmik Gohil

A premium, interactive developer portfolio website built using:
- **React.js** (Vite)
- **Tailwind CSS v4**
- **Lucide Icons**
- **HTML5 Canvas** (3D Radar Skill Profile & Interactive comparison charts)

## Getting Started

To run the project locally:

1. Navigate to the project directory:
   ```bash
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
